# SVA — Unofficial Audio Visualizer

SVA is an unofficial browser extension that provides real-time loudness and spectrum visualization for audio playback when used on the Suno website.

## Features

• Real-time LUFS measurement  
• True Peak detection  
• Loudness Range analysis  
• Frequency spectrum visualization  
• Stereo phase visualization  
• Session-based snapshot comparison  
• Japanese / English UI support
• Panel copy — capture the panel as a PNG image to clipboard


## Privacy

• No audio data is recorded  
• No data is uploaded or transmitted  
• All processing occurs locally  
• Snapshot data exists only in memory  
• Only UI language preference is stored locally  
• The copy button captures the panel locally as a PNG and writes it to your clipboard. No data is transmitted externally.
